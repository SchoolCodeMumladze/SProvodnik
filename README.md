# SПроводник - первый на Python
SПроводник написан на языке программирования Python. Существующие функции:
1. Открытие файла/папки.
2. Удаление файла/папки (работает только при предоставлении соответствующего разрешения!).
3. Переименование файла/папки (работает только при предоставлении соответствующего разрешения!).
4. Создание файла/папки.
В перспективе:
1. Перемещение файла.
2. Коприрование файла.
Для работы с файлами была задействована библиотека OS, а для создания графического интерфейса - TKinter.
