# SПроводник
SПроводник - проект простого, но понятного и удобного проводника.
# SПроводник - первый на Python
SПроводник написан на языке программирования Python. Существующие функции:
1. Открытие файла/папки.
2. Удаление файла/папки (работает только при предоставлении соответствующего разрешения!).
3. Переименование файла/папки (работает только при предоставлении соответствующего разрешения!).
4. Создание файла/папки.
5. Поиск файла/папки.
В перспективе:
1. Перемещение файла.
2. Коприрование файла.
Для работы с файлами была задействована библиотека OS, а для создания графического интерфейса - TKinter.
# Когда пригодится SПроводник?
Программа понадобится, когда ты не понимаешь, как работать с файлами в обычном проводнике - интерфейс SПроводника максимально понятен!
# Как пользоваться SПроводником?
При запуске программы вас встречает основное окно. В нем вы можете:
 - Написать название нового файла/папки и создать его.
 - Включить поиск файла/папки в текущей директории.
 - Нажать на одну из кнопок ниже и перейти к желаемому файлу/папке.
При нажатии на одну из кнопок с припиской Файл/Папка появляется дополнительное окно, в котором вам предлагают разные действия с выбранным элементом. После этого просто закройте дополнительное окно, и всё пошло по новой!
Если вы поняли, что написано выше, то знайте - вы уже умеете пользоваться SПроводником.
